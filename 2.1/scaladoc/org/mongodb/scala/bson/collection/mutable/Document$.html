    The mongodb scala driver.

    To get started you need a MongoClient instance, either from a connection string or via a org.mongodb.scala.MongoClientSettings.

    Notable packages include:

    • MongoClient The MongoClient used to connect and access MongoDB
    • MongoDatabase Providing access to a specific database
    • MongoCollection Providing access to a specific collection in a database.
